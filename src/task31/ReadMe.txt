Для создания сервиса бонусов была предложена следующая схема:

- Выбирается целое число n.
- В помеченные ячейки матрицы n x n записываются n различных чисел от 1 до n^2.
- Остальные (n^2 - n) ячеек остаются пустыми.
- Пользователь получает бонус, если угадывается числа, расположенные в помеченных ячейках.

Для получения бонуса нужно заполнить матрицу n x n таким образом, 
чтобы все числа от 1 до n^2 встречались ровно один раз, 
а во всех помеченных ячейках числа совпадали с выигрышным шаблоном

Найдите любую выигрышную матрицу.

Формат ввода:

В первой строке входных данных записано целое число n(2 <= n <= 100).
Далее в n строках записаны числа в матрице-шаблоне a[i,j] (0 <= a[i,j] <= n^2)
Если a[i,j] = 0, то соответствующая ячейка матрицы не является помеченной и должна
быть заполнена. Если a[i,j] != 0, то в соответствующую ячейку матрицы нужно вписать
a[i,j].

Формат вывода:

Выведите n строк по n целых чисел - любую из выигрышных матрицы.
Гарантируется, что существует как минимум одна выигрышная матрица.

Ограничение времени: 1с
Ограничение памяти: 256 МБ

Пример 1
Ввод:		Вывод:	
4
0 0 0 0		6 16 14 11
1 2 3 4		1 2 3 4
0 0 0 0		8 5 7 10
0 0 0 0		13 12 9 15

Пример 2
Ввод:		Вывод:	
4
1 0 0 0		1 2 4 3
0 6 0 0		14 6 8 5
0 0 11 0	7 10 11 13
0 0 0 16	12 9 15 16
